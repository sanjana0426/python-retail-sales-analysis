📊 Retail Sales Analysis (Python & Power BI)
🔧 Tools Used:
Python (Pandas, NumPy, Matplotlib, Seaborn)
Power BI
📁 Dataset: Retail sales dataset including revenue, supplier, product categories

🔍 Analysis Performed:
Data cleaning and preprocessing
Feature engineering (Total Sales, Date, Month)
Exploratory Data Analysis (EDA)

📈 Key Insights:
Peak sales observed in July
Warehouse sales higher than retail
Beer category highest contributor
